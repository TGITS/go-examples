# AoC 2023 Jour 7

**Version anglaise disponible :** [README.md](README.md)

## Aperçu

Solution Go pour [Advent of Code 2023 - Jour 7](https://adventofcode.com/2023/day/7) (Camel Cards), portée depuis une implémentation TypeScript/Deno.

Contenu actuel :
- Point d'entrée (`main.go`)
- Package solver (`camelcards/`)
- Tests unitaires dans `camelcards/solver_test.go`
- Fichiers d'entrée dans `data/`

## Exécution

Depuis ce dossier (`aoc/2023/aoc20231207`) :

```powershell
go run .
```

## Tests

Depuis ce dossier (`aoc/2023/aoc20231207`) :

```powershell
go test ./...
```

## Fichiers d'entrée

- `data/input_test.txt`
- `data/input.txt`
  - Ce fichier est spécifique à chaque utilisateur AoC et n'est pas versionné dans le dépôt : vous devez fournir votre propre fichier d'entrée.
