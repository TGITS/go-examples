# AoC 2023 Day 7

## Overview

Go solution for Advent of Code 2023 - Day 7 (Camel Cards), ported from a TypeScript/Deno implementation.

Current content:
- Entry point (`main.go`)
- Solver package (`camelcards/`)
- Unit tests in `camelcards/solver_test.go`
- Input files under `data/`

## Run

From this folder (`aoc/2023/aoc20231207`):

```powershell
go run .
```
