# AoC 2023 Day 7

**French version available:** [README_FR.md](README_FR.md)

## Overview

Go solution for [Advent of Code 2023 - Day 7](https://adventofcode.com/2023/day/7) (Camel Cards), ported from a TypeScript/Deno implementation.

Current content:
- Entry point (`main.go`)
- Solver package (`camelcards/`)
- Unit tests in `camelcards/solver_test.go`
- Input files under `data/`

## Run

From this folder (`aoc/2023/aoc20231207`):

```powershell
go run .
```

## Test

From this folder (`aoc/2023/aoc20231207`):

```powershell
go test ./...
```

## Input files

- `data/input_test.txt`
- `data/input.txt`
  - This file is specific to each AoC user and is not committed to the repository: you must provide your own input file.
